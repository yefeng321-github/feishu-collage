# 🖼 飞书多维表格图片拼图插件

一个飞书多维表格自定义侧边栏插件，支持自动读取附件字段中的图片，自动拼接成网格图并保存回指定附件字段，同时提供手动画布编辑功能。
使用场景：在使用VEO制作AI漫剧时，由于该视频模型只支持最多3张参考图，这时可以使用该插件将图片拼接成一张参考图。

## ✨ 功能特性

- **自动拼接**：点击含图片的行，自动加载图片并按网格排列，无需手动操作
- **智能网格**：根据图片数量自动计算布局（1张=1格，2张=2格，3-4张=2×2，5-6张=3×2，以此类推）
- **保持原比例**：图片居中显示，不裁剪，不拉伸，空白处填充白色背景
- **画布固定1:1**：输出图片始终为正方形（1200×1200）
- **自动保存**：拼接完成后自动写入目标附件字段
- **手动编辑**：支持拖拽、缩放、旋转、调整层级、添加文字
- **智能判断**：点击目标字段列时不触发自动拼接，避免循环操作

## 🚀 部署方式

### 1. Fork 或上传到 GitHub

将本项目上传到你的 GitHub 仓库。

### 2. 部署到 Vercel

1. 登录 [vercel.com](https://vercel.com)，新建项目，连接 GitHub 仓库
2. 进入项目 **Settings → General → Build & Development Settings**，设置：
   - **Build Command**：Override 打开，填一个空格
   - **Install Command**：Override 打开，填一个空格
   - **Output Directory**：Override 打开，填 `.`
3. 点击 **Save**，然后 **Redeploy**
4. 部署完成后获得访问地址，如 `https://feishu-collage.vercel.app`

### 3. 在飞书多维表格中添加插件

1. 打开飞书多维表格
2. 点击右上角 **插件** → **添加插件** → **自定义插件**
3. 填入 Vercel 部署地址
4. 保存，侧边栏中即可看到插件

## 📖 使用说明

1. 在多维表格中确保有**两个附件字段**：一个存原始图片，一个用于保存拼接结果
2. 打开插件侧边栏，选择**来源字段**和**保存结果到字段**
3. 点击表格中含图片的行，插件自动完成拼接并保存
4. 如需手动调整，点击 **🎨 手动编辑画布** 进行自定义编排

## 🛠 技术栈

- 飞书多维表格 JS SDK（`@lark-base-open/js-sdk`）
- 原生 HTML / CSS / JavaScript，无框架依赖
- Canvas API 用于图片合成
- Vercel 静态托管

## 📝 注意事项

- 需要开启 VPN 才能访问 Vercel 部署地址（国内网络限制）
- 图片临时 URL 有效期为 10 分钟，请及时完成拼接
- 建议来源字段和目标字段使用不同的附件字段，避免覆盖原图
